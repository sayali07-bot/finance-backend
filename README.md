# Finance Tracker API

## 📌 Project Overview
This is a simple backend API for managing personal finance records.  
Users can register, login, and manage income & expense records.

## 🚀 Setup Instructions

1. Clone the repository
git clone https://github.com/sayali07-bot/finance-backend.git
cd finance_back


2. Install dependencies
    npm install


3. Start server
    npx nodemon app.js


Server runs on:
http://localhost:5000/
